# ConTuCuota Web - FINAL

Este es el paquete final para desplegar en producción el sitio web de ConTuCuota.

## Instrucciones

1. Sube todo el contenido al mismo directorio raíz de tu servidor.
2. Asegúrate de que `index.html`, `styles.css`, `favicon.ico`, `logo_contucuota_nuevo.png`, y `hero-background.jpg` estén visibles desde la web.
3. Verifica que el navegador no esté cargando contenido en caché (Ctrl+F5).

## Funcionalidades incluidas

- Secciones completas y visuales
- Simulador Chart.js
- Calculadora IRPF funcional
- Estilo responsive
- Favicon compatible
